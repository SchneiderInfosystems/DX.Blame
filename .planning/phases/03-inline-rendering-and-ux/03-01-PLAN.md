    **DX.Blame.Formatter.pas:**
    Create pure functions (no OTA dependency except DeriveAnnotationColor):

    1. `function FormatRelativeTime(ADateTime: TDateTime): string;`
       Uses System.DateUtils (YearsBetween, MonthsBetween, DaysBetween, HoursBetween, MinutesBetween).
       Returns "N years/months/days/hours/minutes ago" or "just now".
       Uses System.StrUtils.IfThen for singular/plural.

    2. `function FormatBlameAnnotation(const ALineInfo: TBlameLineInfo; const ASettings: TDXBlameSettings): string;`
       - If IsUncommitted, return cNotCommittedAuthor
       - Build string: [Author, ] [time] [ bullet Summary]
       - Truncate with ellipsis (Unicode #$2026) if exceeds MaxLength
       See RESEARCH.md Example 2 for reference implementation.

    3. `function DeriveAnnotationColor: TColor;`
       - Query INTACodeEditorServices.Options.BackgroundColor[atWhiteSpace]
       - Blend 40% toward mid-gray (128,128,128)
       - Fallback to clGray if services unavailable
       See RESEARCH.md Pattern 4 for algorithm.
