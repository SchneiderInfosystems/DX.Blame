  <files>src/DX.Blame.Hg.Discovery.pas</files>
  <action>
Add a new public function `FindThgExecutable` to DX.Blame.Hg.Discovery.pas:

1. In the interface section, add after `ClearHgDiscoveryCache`:
   ```pascal
   /// <summary>
   /// Returns the full path to thg.exe (TortoiseHg GUI launcher), or an empty
   /// string if not found. Derives the path from hg.exe since both ship in the
   /// same TortoiseHg installation directory.
   /// </summary>
   function FindThgExecutable: string;
   ```

2. In the implementation, add after `ClearHgDiscoveryCache`:
   ```pascal
   function FindThgExecutable: string;
   var
     LHgPath: string;
   begin
     LHgPath := FindHgExecutable;
     if LHgPath = '' then
       Exit('');
     Result := TPath.Combine(TPath.GetDirectoryName(LHgPath), 'thg.exe');
     if not TFile.Exists(Result) then
       Result := '';
   end;
   ```

This leverages the already-cached hg.exe path from FindHgExecutable. No separate caching needed since FindHgExecutable itself is cached. TPath and TFile are already available via System.IOUtils in the uses clause.

4. Add a helper procedure to launch thg.exe fire-and-forget:
   ```pascal
   procedure LaunchThg(const ACommand, ARepoRoot, AFilePath: string);
   var
     LThgPath: string;
     LArgs: string;
   begin
     LThgPath := FindThgExecutable;
     if LThgPath = '' then
       Exit;
     LArgs := ACommand + ' -R "' + ARepoRoot + '" "' + AFilePath + '"';
     ShellExecute(0, 'open', PChar(LThgPath), PChar(LArgs), PChar(ARepoRoot), SW_SHOWNORMAL);
   end;
   ```

5. Implement the click handlers. Both get the current file from the editor:
   ```pascal
   procedure TNavigationMenuHandler.OnThgAnnotateClick(Sender: TObject);
   var
     LEditorServices: IOTAEditorServices;
     LTopView: IOTAEditView;
     LFileName: string;
   begin
     if not BlameEngine.VCSAvailable then
       Exit;
     if not Supports(BorlandIDEServices, IOTAEditorServices, LEditorServices) then
       Exit;
     LTopView := LEditorServices.TopView;
     if LTopView = nil then
       Exit;
     LFileName := LTopView.Buffer.FileName;
     LaunchThg('annotate', BlameEngine.RepoRoot, LFileName);
   end;

   procedure TNavigationMenuHandler.OnThgLogClick(Sender: TObject);

     if LTopView = nil then
       Exit;
     LFileName := LTopView.Buffer.FileName;

   end;
   ```
