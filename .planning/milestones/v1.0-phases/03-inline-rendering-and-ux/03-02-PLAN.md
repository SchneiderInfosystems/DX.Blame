  <action>
    **DX.Blame.Renderer.pas:**
    Create TDXBlameRenderer implementing INTACodeEditorEvents and INTACodeEditorEvents370 (from ToolsAPI.Editor).
    Inherit from TNotifierObject for IUnknown/IOTANotifier defaults.

    Key implementation details:

    1. **AllowedEvents:** Return [cevPaintLineEvents, cevCaretEvents] -- PaintLine for rendering, CaretEvents for tracking current line.

    2. **AllowedLineStages:** Return [plsEndPaint] -- paint after IDE finishes.

    3. **EditorSetCaretPos (INTACodeEditorEvents370):** Track FCurrentLine := Y (1-based line number) and FCurrentEditor := Editor. Call InvalidateTopEditor to trigger repaint when caret moves (so annotation follows cursor in dsCurrentLine mode).

    4. **PaintLine implementation:**
       - Guard: Exit if (Stage <> plsEndPaint) or BeforeEvent or not BlameSettings.Enabled.
       - Get logical line number from Context.LogicalLineNum (1-based).
       - Display scope check: If BlameSettings.DisplayScope = dsCurrentLine, only paint when Context.LogicalLineNum = FCurrentLine.
       - Get the current file path from Context.EditView (IOTAEditView) -- use Context.EditView.Buffer.FileName.
       - Look up blame data: BlameEngine.Cache.TryGet(LFileName, LBlameData).
       - Index into LBlameData.Lines array: LLineIndex := LLogicalLine - 1 (array is 0-based, LogicalLineNum is 1-based). Bounds check.
       - Format annotation: LText := FormatBlameAnnotation(LBlameData.Lines[LLineIndex], BlameSettings).
       - If LText is empty, exit.
       - Save canvas state (Font, Brush) in local vars. Use try/finally to restore.
       - Set font: LCanvas.Font.Style := [fsItalic]. Keep same font name/size as editor (already set on canvas).
       - Set color: If BlameSettings.UseCustomColor then BlameSettings.CustomColor else DeriveAnnotationColor.
       - Compute X position: Context.LineState.VisibleTextRect.Right + (Context.CellSize.cx * 3) for 3 chars padding.
       - Set brush style to bsClear (transparent background for text).
       - TextOut at (LAnnotationX, Rect.Top).
       - Restore canvas state in finally block.

    5. **AllowedGutterStages:** Return [] (no gutter painting).
    6. **UIOptions:** Return [] (no UI option flags needed).
    7. **All other INTACodeEditorEvents methods:** Empty stubs (required by interface).
    8. **All INTACodeEditorEvents370 methods except EditorSetCaretPos:** Empty stubs.

    Module-level registration helpers:
    - `procedure RegisterRenderer;` -- Supports(BorlandIDEServices, INTACodeEditorServices, LServices) then GRendererIndex := LServices.AddEditorEventsNotifier(TDXBlameRenderer.Create)
    - `procedure UnregisterRenderer;` -- if GRendererIndex >= 0 then LServices.RemoveEditorEventsNotifier(GRendererIndex)

    IMPORTANT pitfall avoidance (from RESEARCH.md):
    - Do NOT use INTAEditViewNotifier (deprecated). Use INTACodeEditorEvents.
    - Save and restore canvas state to prevent corruption of IDE painting.
    - Use Context.LogicalLineNum (not EditorLineNum) for correct folded code handling.
    - Keep PaintLine fast -- only dictionary lookup + TextOut, no heavy computation.

    **DX.Blame.KeyBinding.pas:**
    Create TDXBlameKeyBinding implementing IOTAKeyboardBinding (from ToolsAPI).
    Inherit from TNotifierObject.

    Implementation per RESEARCH.md Pattern 3:
    - GetBindingType: Return btPartial
    - GetName: Return 'DX.Blame.ToggleBlame'
    - GetDisplayName: Return 'DX Blame Toggle'
    - BindKeyboard: BindingServices.AddKeyBinding([ShortCut(Ord('B'), [ssCtrl, ssAlt])], ToggleBlame, nil)
    - ToggleBlame: Toggle BlameSettings.Enabled, call BlameSettings.Save, call InvalidateAllEditors, set BindingResult := krHandled

    Module-level helpers:
    - `procedure RegisterKeyBinding;` -- GKeyBindingIndex := (BorlandIDEServices as IOTAKeyboardServices).AddKeyboardBinding(TDXBlameKeyBinding.Create)
    - `procedure UnregisterKeyBinding;` -- if GKeyBindingIndex >= 0 then (BorlandIDEServices as IOTAKeyboardServices).RemoveKeyboardBinding(GKeyBindingIndex)

    Helper procedure (can be in Renderer or KeyBinding):
    - `procedure InvalidateAllEditors;` -- Supports(BorlandIDEServices, INTACodeEditorServices, LServices) then LServices.InvalidateTopEditor

    All .pas files: UTF-8 with BOM, CRLF line endings, standard MIT header.
